Working with Manuscripts

Creating a new manuscrupt

To create a new manuscript, choose 'New Manuscript' from the file menu. Manuscripts can also be created from a folder document view.

The manuscript editor pane

A - Scene List

Shows all the scenes in the manuscript. Displays their name, tags, and wordcount.

Click the empty space in the 'C' column to choose the scene's color.

B - Scene Filter

Type in this box to filter scenes by tag.

The X button clears the current filter.

The refresh button re-applies the filter, in case scene tags have changed.

C - Collapse Panes

Use these buttons to show or hide the scene list or scene editor.

D - Scene Editor

This text editor has the same features as a text document editor.

Adding scenes to a manuscript

Right click on the scene list and choose 'new scene'. The new scene is added after the selected scene or at the end of the manuscript.

Renaming scenes

Find the scene in the scene list. Click the name to begin renaming the scene.

Reordering scenes

Click and drag scenes to reorder them. The scene will be reinsterted after the scene it is dropped on.

Tagging and filtering scenes

Edit a scene's tag text in the scene editor, using the text box directly below the scene's title.

Use the filter box to display only those scenes that contain the text within their tag text.

Manuscript word count

You can take the word count of a manuscript the same as any other document.

Only the words actually in the scene's prose are counted. Names and tags are ignored.

Manuscripts on disc

Manuscripts are stored as JSON. This is a plain-text format, which means your data is never locked behind a binary barrier. While the text may not be immediately useable (as your prose will be heavily escaped to support the JSON format), your manuscript can be recovered from this file with tools no more sophisticated than Notepad.
